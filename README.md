# 彰濱癌症中心 - 癌症資源中心管理系統

管理營養物資、假髮租借、活動記錄、個案及轉介的整合系統。

**當前版本：V1.6**

## ⚠️ 重要警告

1. **資料存在瀏覽器**：清除瀏覽器資料會遺失所有資料！
2. **iOS 7 天清除**：iOS Safari 會在 7 天未使用後自動清除網站資料
3. **必須定期備份**：點選首頁備份按鈕，儲存到 iCloud / Google Drive / 電腦

## 📋 版本歷史

### V1.6 (2026-03-22)
- 版本號移至 Logo 旁獨立顯示，不再重疊
- 物資小卡依品項自動配色（柔色系，6 種色調循環）
- 統計新增物資進貨批次、發放總量、捐贈來源排行、發放對象分布
- 統計新增假髮租借長度分布、區間歸還數、區間帳目收支
- 測試資料擴充：8 位個案（5 種癌別）、11 批物資（含多批次同品項）、5 筆租借、8 筆現金流、7 場活動、8 筆轉介

- 物資庫存改為三欄小卡片，一列顯示三個品項
- 到期預警天數可在設定頁自訂（預設30天）
- 個案管理加入癌別快速篩選列
- 統計報表加入月/季/年/自訂區間篩選，數字依區間動態更新

- 版本號顯示於首頁 Logo 旁

- 物資庫存改為品項分組，點選展開各批到期日明細
- 假髮庫存卡片改為緊湊小卡設計

### V1.3 (2026-03-22)
- 版本號顯示於設定頁底部，含完整版本歷史

### V1.2 (2026-03-21)
- 假髮改為短/中/長庫存量管理
- 假髮新增帳目現金流（收入/支出/結餘）

- 「活動」Tab 改為「報表」（活動記錄＋統計報表）

## 🚀 使用方式

### 方式一：使用 npm（推薦）

```bash
cd /path/to/cancer-resource-center
npm install
npm run dev
```

### 方式二：Python 簡易伺服器

```bash
cd dist && python3 -m http.server 8080
```

## 📱 手機測試

```bash
npm run dev -- --host
```
