# 彰濱癌症中心 - 癌症資源中心管理系統

**當前版本：V1.9**

## ⚠️ 重要警告
1. **資料存在瀏覽器**：清除瀏覽器資料會遺失所有資料！
2. **iOS 7 天清除**：iOS Safari 會在 7 天未使用後自動清除
3. **必須定期備份**：首頁備份按鈕 → 儲存到 iCloud / Google Drive / 電腦
